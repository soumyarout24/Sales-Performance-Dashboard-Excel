# Sales Performance Dashboard & Customer Behavior Analysis

## 📌 Project Overview
This project is an interactive Sales Performance Dashboard developed using **Advanced Excel** to analyze revenue trends, customer behavior, regional sales performance, and product profitability.

The dashboard provides business insights through interactive charts, Pivot Tables, Pivot Charts, slicers, and scenario analysis techniques to support strategic decision-making and performance monitoring.

---

## 🚀 Key Features
- Interactive Excel dashboard with slicers and filters
- KPI tracking for:
  - Total Revenue
  - Total Orders
  - Average Order Value
- Revenue analysis by:
  - Region
  - Product Category
  - Customer Trends
  - Monthly & Yearly Performance
- Top-performing products analysis
- Revenue distribution analysis
- Customer behavior analysis
- Multi-scenario business analysis
- Dynamic data visualization using Pivot Charts
